Event Tags is a plugin for wordpress which generates a link to a Last.FM event page from machine-tagged posts.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `event_tags` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `<?php event_tags(); ?>` in your templates
